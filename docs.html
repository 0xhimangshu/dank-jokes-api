Jokes API Documentation

Introduction

Welcome to the Jokes API documentation. This API provides access to a collection of jokes in different languages and categories.

Base URL

http://104.197.122.48:6901/

Endpoints

1. Root Endpoint

GET /

Returns a welcome message.

Example Response:

{
    "message": "Hello, World!"
}

2. Random Hindi Joke

GET /random/hindi

Returns a random joke in Hindi language.

Example Response:

{
    "joke": "joke text here",
    "author": "author name",
    "category": "category name",
    "language": "Hindi"
}
